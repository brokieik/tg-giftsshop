# 🎁 Gift Shop — Railway Deployment

  ## Деплой на Railway

  1. Загрузи папку на GitHub
  2. railway.app → New Project → Deploy from GitHub
  3. Environment Variables:
     - BOT_TOKEN
     - ADMIN_ID
     - WEBAPP_URL (вставь после деплоя)
  4. После деплоя скопируй URL → вставь в WEBAPP_URL → Redeploy

  ## Локальный запуск
  npm run build
  npm start
